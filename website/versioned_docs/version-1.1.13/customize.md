## Customize

<p>You can customize the predefined variables and colors when initializing Bootnative.</p>

```javacript
const bn = bootnative({
    vars, colors
});
```

## Default vars
``` javascript
{
    spacing: 5, //margins & pading (scale prop)
    container_mt: 0, //mt -> margin top
}
```
